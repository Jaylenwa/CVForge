# 内容预设规范
- 所有内容预设需遵循统一的结构与写作规范，保证可用性与一致性。
- 各模块按岗位灵活组合；并非所有模块都必须出现，但出现的模块必须完整、逻辑自洽。
- 语言需与预设的 Language 一致（zh 或 en），字段内容做本地化处理。

## 顶层结构
- title：简历标题（本地化）
- language：zh|en
- Personal：个人信息对象（见下）
- Theme：主题配置对象（见下）
- sections：模块数组（每个元素为一个模块对象）

## Personal 字段
- FullName：姓名
- Email：邮箱
- Phone：手机号（字符串）
- AvatarURL：头像 URL（可留默认 /avatar.avif）
- Job：岗位/目标岗位（与简历方向一致）
- City：城市
- Money：期望薪资（字符串，如 25k-35k）
- JoinTime：到岗时间（如 随时到岗 / Available immediately）
- Gender：性别（zh：男/女；en：Male/Female）
- Age：年龄（字符串）
- Degree：最高学历（zh：本科/硕士/博士；en：Bachelor/Master/PhD）
- CustomInfo：JSON 字符串形式的自定义信息数组：[{"label":"…","value":"…"}]

## Theme 字段
- Color：HEX 颜色值（如 #2563eb）
- Font：字体标识（notosans）
- Spacing：间距（normal）
- FontSize：字号（如 13）

## sections 模块对象
- id：模块唯一标识（字符串，文件内唯一）
- type：模块类型，使用以下规范化枚举的小写值：
  - summary, experience, education, skills, projects, internships, portfolio, awards, interests, exam, selfevaluation, custom
- title：模块标题（本地化）
- isVisible：是否显示（布尔）
- items：条目数组（见下）

## items 条目对象
- id：条目唯一标识（字符串，模块内唯一）
- Title：主标题（公司/学校/项目等）
- Subtitle：副标题（岗位/方向等；务必为实际岗位，如“Java 后端开发工程师”）
- Major：专业（教育模块常用）
- Degree：学位/学历（教育模块常用）
- TimeStart：开始时间（YYYY-MM）
- TimeEnd：结束时间（YYYY-MM）
- Today：是否至今（布尔）
- Description：详情描述，建议使用 HTML 列表：
  - 使用 <ul><li>…</li></ul>，每个 li 描述一个要点
  - 强调成果与量化指标（如 QPS、延迟、用户数、转化率等）

## 写作规范
- 摘要（summary）：1 条目即可，聚焦优势与关键词能力，用 <ul><li> 列表呈现。
- 工作经历（experience）：Title=公司，Subtitle=岗位，包含时间与成果；描述聚焦职责、技术栈、量化结果。
- 项目经历（projects）：Title=项目名，Subtitle=岗位，包含时间；描述需包含技术栈、关键设计/优化与量化结果。
- 技能清单（skills）：按主题分组（如“语言与框架”“存储与中间件”“工程化与稳定性”），Description 以逗号分隔的关键技能。
- 教育背景（education）：Title=学校，Major/ Degree 必填，包含时间与概述；满足学历层级逻辑（见下）。
- 作品（portfolio）：仅包含 Description，使用 <a> 链接，但域名必须为不可解析的虚域（如 *.invalid）。
- 其它模块（internships/awards/interests/exam/selfevaluation/custom）：按上述字段规范填充，保持描述真实、可验证。

## 教育层级逻辑
- 若最高学历为硕士，则必须至少包含一个本科条目。
- 若最高学历为博士，则必须至少包含一个硕士与一个本科条目。

## 隐私与合规
- 不使用真实个人隐私信息（真实手机号、身份证、家庭住址等）。
- 所有外链需使用不可解析域（如 code.zhwei.invalid / blog.zhwei.invalid / npm.zhwei.invalid）。
- 时间采用 YYYY-MM；进行中的条目设置 Today=true。

## 质量要求
- 内容真实、准确、逻辑清晰；避免一句话概述。
- 面向岗位定制，确保 Subtitle 与岗位匹配，描述包含可量化成果。
- 链接与隐私合规；教育层级满足前述约束。
